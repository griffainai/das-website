LOGO FILE INSTRUCTIONS
======================
Place the logo file here as: logo.png (or logo.svg)

The logo is referenced in the HTML as:
  <img src="images/logo.png" alt="Driver Appreciation Solutions" class="nav-logo-img">

A white pill background is applied via CSS so the navy/silver logo
displays correctly on the dark navigation bar.

If you have a white/transparent version of the logo designed for dark
backgrounds, remove the CSS rule for .nav-logo-img background/padding.
